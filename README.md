# Cisco Webex webhooks examples

## About

Collection of demo scripts for collecting data and pushing it to a Cisco Webex space via webhook.

---

## Scripts

**CP_Cluster_Status.sh**
Bash script for retrieving HA status from two Check Point firewall nodes

**Current_IP.sh**
Bash script for retrieving current IP

**Temp_and_humidity.sh**
Bash script for retrieving temperature + humidity from DHT22 on RPi
